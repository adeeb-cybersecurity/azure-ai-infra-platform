# Azure AI Infrastructure Platform

A cloud-native infrastructure platform on Azure built with Terraform, GitHub Actions, Docker, CI/CD, and observability tooling.

This project is designed as a production-style foundation for hosting future AI workloads such as Azure AI Foundry or Azure OpenAI services.

## Goals

- Provision Azure infrastructure using Terraform
- Build and deploy containerized services
- Automate validation and deployment with GitHub Actions
- Add observability with logs, metrics, and dashboards
- Document architecture and operational decisions

## Tech Stack

- Azure
- Terraform
- GitHub Actions
- Docker
- Azure Container Registry
- Azure Container Apps
- Azure Monitor / Log Analytics
